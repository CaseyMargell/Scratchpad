# Scratchpad

A lightweight spreadsheet scratchpad. Quick-launch, no bloat.

## Use it

Visit [https://caseymargell.github.io/Scratchpad/scratchpad.html](https://caseymargell.github.io/Scratchpad/scratchpad.html) in Edge or Chrome and click "Install this site as an app" to add it to your Start menu.

## Features

- 6x16 cell grid with resizable columns
- Formulas: SUM, AVERAGE, MIN, MAX, COUNT, COUNTIF, SUMIF, IF, ROUND, ABS, SQRT, POWER, MOD, CONCATENATE, UPPER, LOWER, TRIM, LEN, LEFT, RIGHT, TODAY, NOW
- Click-drag or arrow-key cell references in formulas (Excel-style)
- Save as .xlsx, open .xlsx
- Dark/light mode
- Undo/redo (Ctrl+Z/Y, 50 steps)
- Copy/paste compatible with Excel
- Auto-saves to local storage
- Works offline once loaded
